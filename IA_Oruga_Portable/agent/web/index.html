🐛 Consola de IA Oruga

¿Qué es IA Oruga?

Es un sistema local de agentes autónomos diseñado para ciencia de datos que puede invocar herramientas (skills) para analizar CSVs, código Python o ejecutar notebooks de Jupyter.

Generador de Tareas Autónomas

Comando de Ejecución Local:

./start-oruga.sh --goal ...

Ejecuta este comando en tu terminal para iniciar el agente de forma autónoma con todos tus prompts y skills en este proyecto.
